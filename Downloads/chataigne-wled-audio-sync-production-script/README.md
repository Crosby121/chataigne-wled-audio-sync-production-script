
# chataigne-wled-audio-sync-production-script

Automates setup for WLED + Chataigne audio sync production environment on Windows 11. Includes:
- Voicemeeter Potato installation & routing
- Sonic Annotator + Vamp plugins
- Chataigne modules (SCAnalyzer, WLEDAudioSync)
- MIDI and VBAN configuration
- Automatic backup and cleanup
